# Lawer — AI-ассистент юридического отдела

Автоматизация рутинных задач юристов: анализ и генерация документов, правовая аналитика РФ, ведение сроков и задач. Цель — сократить время на типовые операции и повысить единообразие юридических документов.

**Юрисдикция:** Российская Федерация.
**Полное техническое задание:** [TZ.md](./TZ.md)

## Технологический стек

| Слой | Технологии |
|---|---|
| Frontend | Next.js 14+, React 18, TypeScript, Tailwind CSS, Zustand |
| Backend | Node.js 20+, TypeScript, REST + SSE, Prisma |
| СУБД | PostgreSQL 16 + pgvector |
| Очереди | Redis + BullMQ |
| Хранение файлов | MinIO (S3-совместимое) |
| LLM | Claude API (Anthropic) |
| MCP | @modelcontextprotocol/sdk |
| Контейнеризация | Docker Compose |

## Структура репозитория

```
law-ai/
├── backend/        # API-сервер (Node.js + TypeScript + Prisma)
├── frontend/       # веб-интерфейс (Next.js)
├── mcp-servers/    # MCP-серверы (инструменты для LLM)
├── docker/         # Dockerfile'ы отдельных сервисов
├── .claude/        # конфиги специализированных агентов Claude Code
├── .env.example    # шаблон переменных окружения
└── TZ.md           # техническое задание
```

## Запуск

Проект в активной разработке. Инструкции по запуску появятся после готовности Docker Compose.

Предварительно:

```bash
cp .env.example .env
# заполнить значения в .env
```

## Фазы разработки

1. **MVP** (текущая) — чат со стримингом, аутентификация, анализ и генерация документов, базовые MCP-серверы.
2. **Правовая аналитика** — RAG по нормативной базе РФ, анализ судебной практики.
3. **Задачи и база знаний** — реестр сроков, пополняемая KB, календарь процессуальных сроков.
4. **Hardening** — RBAC, аудит-лог, security-тестирование, пользовательская документация.

Подробнее — в [TZ.md](./TZ.md).

## Принципы разработки

- **1 задача = 1 коммит.** Чистая история, коммиты подписываются агентом-исполнителем.
- **Специализированные агенты.** Задачи делегируются агентам (`backend-developer`, `frontend-ui-developer`, `docker-deploy`, `devops-ci`, `qa-engineer`, `security-auditor`, `llm-engineer`).
- **Всё на русском.** Комментарии, объяснения, сообщения коммитов — на русском языке.
- **Безопасность по умолчанию.** Данные на территории РФ (152-ФЗ), запрет передачи конфиденциальной информации во внешние LLM без явного согласия.
